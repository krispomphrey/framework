create_user()

create_user(array $data) : \array/boolean

Implements create_user();

This function helps the app create a standard user. Framework expects a table named users with at least 3 columns: crdate, username, password. The current request time is used, along with the user generated username (checked for letters, hypens periods, underscores and numbers only, no spaces) and a properly hashed password.

Parameters

array $data

An array containing a username => value and password => value pair.

Returns

\array/boolean —

A partial user array, ready to be entered into a database or false.

Uses
$this->generate_password().
